Lesson 1: Integers

Learning Outcome

At the end of this lesson, you should be able to identify integers and represent them on a number line.

What are integers?

Integers are whole numbers that can be:

  • Positive integers: \(1, 2, 3, 4, \dots\)
  • Negative integers: \(-1, -2, -3, -4, \dots\)
  • Zero: \(0\)

Examples of integers:

\(-5,\ -2,\ 0,\ 4,\ 9\)

Why do we need negative numbers?

Negative numbers help us describe values below a reference point.

  • Temperature: −3°C means 3 degrees below zero.
  • Money: −RM20 means owing RM20.
  • Height: −5 m means 5 metres below sea level.

Positive and negative integers

  • Positive integers are written without a sign. Example: \(5\)
  • Negative integers are written with a minus sign (−). Example: \(-5\)

Integers on a number line

On a number line:

  • Numbers increase as we move to the right.
  • Numbers decrease as we move to the left.
  • Zero is the reference point.

Example order:

\(-3 < -1 < 0 < 2 < 5\)

Common Mistake

Mistake: Thinking that \(-8\) is greater than \(-3\) because 8 is greater than 3.

Correct understanding: \(-3\) is greater than \(-8\) because it is closer to zero on the number line.

Mini Practice

  1. List three positive integers.
  2. List three negative integers.
  3. Which is greater: \(-4\) or \(-1\)? Explain.
  4. Arrange \(3, -2, 0, -5\) in ascending order.

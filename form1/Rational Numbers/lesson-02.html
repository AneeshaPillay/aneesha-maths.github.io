Lesson 2: Basic Arithmetic Operations Involving Integers

Learning Outcome

At the end of this lesson, you should be able to add and subtract integers correctly.

Adding integers with the same sign

When adding integers with the same sign:

  • Add their absolute values.
  • Keep the same sign.

Examples:

  • \( (+3) + (+5) = +8 \)
  • \( (-4) + (-6) = -10 \)

Adding integers with different signs

When adding integers with different signs:

  • Subtract their absolute values.
  • Use the sign of the number with the larger absolute value.

Examples:

  • \( (+7) + (-3) = +4 \)
  • \( (-9) + (+5) = -4 \)

Subtraction of integers

Subtracting an integer is the same as adding its opposite.

Rule:

\( a - b = a + (-b) \)

Examples:

  • \( 6 - 4 = 6 + (-4) = 2 \)
  • \( 5 - (-3) = 5 + 3 = 8 \)
  • \( -7 - 2 = -7 + (-2) = -9 \)

Using a number line

  • Start at the first number.
  • Move right for positive numbers.
  • Move left for negative numbers.

Example:

\( -2 + 5 = 3 \)

Start at −2 and move 5 steps to the right.

Common Mistakes

  • Mistake: \(5 - 3 = -2\)
  • Correct: Subtraction does not always give a negative answer.

  • Mistake: \(4 - (-2) = 2\)
  • Correct: Subtracting a negative means adding a positive.

Lesson 3.2: Cubes and Cube Roots

Learning Outcome

At the end of this lesson, you should be able to find cubes and cube roots of whole numbers and identify perfect cubes.

Cube Numbers

A cube number is a number obtained by multiplying a whole number by itself three times.

If the number is \(n\), then its cube is: \[ n^3 = n \times n \times n \]

Examples:

  • \(2^3 = 2 \times 2 \times 2 = 8\)
  • \(3^3 = 27\)
  • \(5^3 = 125\)

Cubing Negative Numbers

When a negative number is cubed, the answer is negative.

  • \((-2)^3 = -8\)
  • \((-5)^3 = -125\)

This is because a negative number is multiplied an odd number of times.

Cube Roots

The cube root of a number is the value that, when multiplied by itself three times, gives the original number.

The cube root symbol is \(\sqrt[3]{\ \ }\).

Examples:

  • \(\sqrt[3]{8} = 2\), because \(2^3 = 8\)
  • \(\sqrt[3]{27} = 3\)
  • \(\sqrt[3]{125} = 5\)
  • \(\sqrt[3]{-64} = -4\)

Perfect Cubes

A perfect cube is a number that can be written as the cube of a whole number.

Examples of perfect cubes:

\(1, 8, 27, 64, 125, 216, 343, 512, 729, 1000\)

Numbers such as 50 or 200 are not perfect cubes.

Try This

  1. Find \(4^3\).
  2. Find \(\sqrt[3]{64}\).
  3. Evaluate \((-3)^3\).
  4. Is 125 a perfect cube? Explain.
  5. Find \(\sqrt[3]{-27}\).

Solutions

  1. \(4^3 = 4 \times 4 \times 4 = 64\)
  2. Since \(4^3 = 64\),
    \(\sqrt[3]{64} = 4\)
  3. \((-3)^3 = -27\)
  4. Yes. \(125 = 5^3\),
    so 125 is a perfect cube.
  5. Since \((-3)^3 = -27\),
    \(\sqrt[3]{-27} = -3\)
